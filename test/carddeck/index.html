The making of this website

3 days ago

This is a static website. And only uses generated HTML5, handcrafted CSS3 and a bit of Javascript. Though with two exceptions:

The first exception is the use of MathJAX to typeset mathematical formula's.

The other exception only applies to source code display and provides syntax colouring for a diversity of programming languages

The HTML5 document structure is generated from XML-documents using XSLT

These XML-documents, most notably, the articles and essays, are written in the Oxygen XML Editor using the "author" mode.
